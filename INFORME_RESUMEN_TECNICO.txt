CONTENIDO SINTETIZADO
====================

Sección 1: Introducción (~1 página)
├─ Contexto: CAGD y elementos finitos
├─ Problema: Interpolación Lagrangiana con puntos de control
├─ Desafío: Matriz de Bernstein-Vandermonde mal condicionada (κ ≈ 10⁶)
├─ Antecedentes: Marco-Martínez (2007) vs. Ainsworth-Sánchez (2015)
└─ Ventajas clave: O(n²), derivación simple, generalizable

Sección 2: Algoritmo Newton-Bernstein (~1.5 páginas)
├─ 2.1 Estrategia: Construcción recursiva mediante forma de Newton
│       p_k = p_{k-1} + w_k(x) f[x_0,...,x_k]
├─ 2.2 Recurrencias: Dos Proposiciones formales
│  ├─ Proposición 1: Coeficientes de Bernstein de w_k(x)
│  │  w_j^{(k)} = (j/k) w_{j-1}^{(k-1)} (1-x_{k-1}) - ((k-j)/k) w_j^{(k-1)} x_{k-1}
│  └─ Proposición 2: Puntos de control de p_k(x)
│     c_j^{(k)} = (j/k) c_{j-1}^{(k-1)} + ((k-j)/k) c_j^{(k-1)} + w_j^{(k)} f[x_0,...,x_k]
├─ 2.3 Complejidad: Demostración de O(n²)
│       ∑_{k=0}^n O(k) = O(n²)
└─ THEOREM: Correctitud del Algoritmo 1 (Ainsworth-Sánchez)

Sección 3: Desempeño y Generalización (~0.8 páginas)
├─ 3.1 Validación Experimental
│   ├─ Tabla comparativa: Newton-Bernstein vs. Marco-Martínez vs. Matlab
│   │   Caso: n=15, κ(A) = 2.3×10⁶
│   │   Datos f_1, f_2, f_3 → Errores relativos en norma L²
│   │   Resultado: Precisión ~10⁻¹⁶ vs. Matlab ~10⁻¹¹
│   └─ Caso Chebyshev: κ(A) = 2.1×10⁷ → Reordenamiento de nodos posible
└─ 3.2 Generalización Multidimensional
    ├─ Producto tensorial 2D: Aplicación recursiva del algoritmo 1D
    ├─ Extensión a 3D sin modificación
    ├─ Interpolación en símplex en ℝ^d: Reducción a secuencia de problemas 1D
    └─ Caso extremo: κ(A_2) = 1.4×10¹³ → Precisión aún superior

Sección 4: Conclusión (~0.2 páginas)
├─ Síntesis: 3 virtudes del algoritmo
│  ├─ Complejidad óptima O(n²)
│  ├─ Estabilidad numérica superior
│  └─ Generalización natural a múltiples dimensiones
├─ Impacto: Facilita uso en análisis de elementos finitos
└─ Perspectivas: Órdenes de Leja, GPUs, splines isogeométricos
